# Conceptos y Hallazgos - Tesis Doctoral

Aplicación web interactiva que presenta los conceptos teóricos, metodologías y hallazgos clave de una investigación doctoral sobre educación ambiental.

## Características

- Navegación intuitiva por categorías
- Sistema de búsqueda en tiempo real
- Modales informativos detallados
- Diseño responsive
- Interfaz en español

## Estructura

- **Conceptos Teóricos**: EPA, Conciencia Ecológica
- **Instrumentos de Medición**: ECA, Validación Psicométrica
- **Metodología**: Diseño Cuasiexperimental, Grupos
- **Hallazgos**: Brecha Actitud-Conducta
- **Intervenciones**: Integración Curricular
- **Enfoques**: Educación Ambiental
- **Contexto**: Semirural

## Tecnologías

- HTML5
- CSS3
- JavaScript Vanilla
- GitHub Pages para despliegue
